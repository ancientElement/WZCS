# 局域网文字传输工具

一个零依赖的局域网文字传输 Web 应用，支持手机、电脑在同一网络下实时互传文字。

## 功能特点

- **零依赖**：纯 Node.js 内置模块实现，无需安装任何 npm 包
- **实时传输**：使用 SSE (Server-Sent Events) 技术，消息实时推送
- **多设备支持**：同一局域网内的任意设备均可连接使用
- **历史记录**：自动保留最近 20 条消息，新设备连接时可查看
- **响应式设计**：适配手机、平板、电脑等各种屏幕

## 快速开始

### 1. 启动服务

```bash
node server.js
```

### 2. 访问应用

服务启动后会显示可访问的地址，例如：

```
========================================
  局域网文字传输工具已启动！
========================================

请用浏览器访问以下地址之一：
  • http://192.168.1.5:3000  (推荐，用于局域网访问)
  • http://localhost:3000

在同一局域网的手机/电脑上打开上述地址即可互传文字
========================================
```

### 3. 开始传输

- 在输入框中输入要发送的文字
- 点击「发送文字」按钮或按 `Ctrl + Enter`
- 其他设备会立即收到消息

## 文件说明

| 文件 | 说明 |
|------|------|
| `server.js` | Node.js 服务端，处理 SSE 连接和消息广播 |
| `index.html` | 前端页面，提供用户界面 |
| `package.json` | 项目配置（无外部依赖） |

## 技术实现

- **SSE (Server-Sent Events)**：服务器向客户端推送实时消息
- **Fetch API**：客户端发送消息到服务器
- **长连接**：每个设备建立持久连接，实现实时通信

## 注意事项

- 所有设备必须在同一局域网内
- 默认端口为 3000，可在 `server.js` 中修改 `PORT` 变量
- 消息仅保存在内存中，服务重启后历史记录会清空
